THE YUFENGTANG PROCESS

從第一句需求，到最後一次驗收

每個階段都有清楚的工作內容與確認節點，讓屋主知道目前進度，也知道下一步要做什麼。

01

需求訪談

了解家庭成員、生活習慣、喜愛風格、收納需求、設備使用與預算方向。

找出真正的設計問題
02

現場丈量

確認尺寸、採光、結構、樑柱、管線、設備與現場限制，建立設計的可靠基礎。

讓方案從現場出發
03

平面規劃

釐清動線與機能分區，提出平面配置與初步預算，討論空間最值得投資的地方。

先把生活安排好
04

設計提案

透過風格、色彩、材質、燈光與 3D 視覺提案，讓抽象想法變成可理解的空間藍圖。

在施工前看見未來
05

合約確認

確認設計範圍、交付圖面、工程項目、付款節點、預算與工期，建立雙方清楚的合作基礎。

透明，是信任的開始
06

細部設計

完成平面、立面、天花、燈具、插座、給排水、櫃體與材料規格等施工依據。

把美感轉成可施工圖面
07

工程管理

依圖施工，進行進度、品質、材料與工種協調，並以現場回報掌握工程狀況。

讓設計品質延續到現場
08

驗收交屋

逐項檢查施工成果、設備功能與細節品質，完成清潔、修正、交屋與售後保固。

把最後一個細節做好

PROJECT TIMELINE

把時間排好，工程自然更安心

實際工期會受到屋況、圖面確認、材料交期、社區施工規範與追加變更影響，以下提供住宅專案的常見節奏。

PHASE 01｜約 1–3 週

需求與設計確認

訪談、丈量、平面規劃、風格提案與預算初估。

PHASE 02｜約 2–5 週

細部設計與發包

施工圖、材料確認、工程報價與合約簽訂。

PHASE 03｜約 2–4 個月

現場施工

拆除、水電、泥作、木作、油漆、設備與家具進場。

PHASE 04｜約 1–2 週

驗收與交屋

細部修正、清潔、功能測試、拍照存檔與保固說明。

建議屋主在開工前預留彈性時間，並將家電、家具、窗簾與特殊訂製品的交期納入整體排程。
HOW TO CHOOSE A DESIGN TEAM

挑選設計公司的三個判斷點

看作品，也看是否適合自己

  • 作品風格是否與期待接近？
  • 是否有相似坪數與屋況的經驗？
  • 能不能理解你的生活，而不只是套用風格？

看報價，更要看內容

  • 設計費與工程費是否分開說明？
  • 材料、數量、工法與付款節點是否清楚？
  • 變更、追加與取消的處理方式是否寫入合約？

看施工管理是否有制度

  • 是否有完整施工圖與工程排程？
  • 由誰負責現場協調與品質檢查？
  • 是否提供進度回報、驗收與保固？

看設計能否長期使用

  • 收納、清潔、維修與設備更新是否被考慮？
  • 燈光與材質是否兼顧美感和日常舒適？
  • 空間能否陪伴家庭未來的變化？
